# LankaLens Research Website

Static HTML/CSS/JavaScript research website ready for GitHub Pages.

## How to use
1. Replace placeholder text with your final project details.
2. Put PDFs inside `assets/docs/` and slides inside `assets/slides/`.
3. Update links in `index.html`.
4. Upload this folder to a GitHub repository.
5. Go to Repository Settings > Pages > Deploy from branch > main > root.
